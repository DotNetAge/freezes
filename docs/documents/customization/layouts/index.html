文稿是通过在元数据中加入 layout 指定使用哪一种部局方式显示文稿内容，主要提供以下几种布局方式

recent

显示站点标题并且将posts目录内的文章从新到旧排列文章, 多用于首页

page

posts 目录以外的文章都将自动使用此布局，此布局按以下顺序显示

  • 页标题
  • 概述 (summary)
  • 页位置导航

  • 页面正文 page.html

  • 本页分类目录下的文章列表

post

显示文章详细内容，站点标题将会被缩小，<h1>显示文章标题，此布局按以下顺序显示

  • 文章标题
  • 封面 (如果有设定)
  • 概述 (summary)
  • 正文 (html)
  • 相关文章 (具有与当前文章相关的tag)
  • 讨论 (集成disgus widget )

base.html

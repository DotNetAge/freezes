You can easy to customize your website layouts and styles without any limitations

You can specified the layout meta to change the page default layout.

默认安装后会提供一个简约的网页风格。每个文稿可以通过元数据 layout 指定当前文稿用什么页面布局显示。默认情况下当文稿内没有采用 layout元数据指定布局时， Freezes 将为你的文稿自动选择一种最合适的布局方式。

Freezes offers 3 default layouts:

Layout Description
recent Show recently posts after the content body.
page Show a sub articles list after the content body.
post Show a disqus comments on the bottom of the page.

For example:

layout: recent
title: "About"

Here is the content body.

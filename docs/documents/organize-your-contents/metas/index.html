Design philosopy

  • A good document should be able to describe itself cleanly.
  • A human readable doucment should not contain nonstandard markups.
  • Do not use the document as a web page template.

Freezes is a tools working with pure markdown documents. So we can use markdown's meta data feature to define the aditional data.

What is meta data ?

Meta data is a group of human readable data on the top of the document body. And the meta data is a common markdown standard so it can be parsed by any Markdown text engines or editos.

For example

title: About me
summary: The text to introduce yourself.
author: Ray

Here is the content.

Freezes can be parsed the metas below：

Meta Description
title Article/Page title text (Require).
description Page description text that render in <meta> Maximum 255 characters.
summary The summary of the article. It also can be use as sub-title
cover Article cover image file url
author Author name.
date Publish date，The format is : yyyy-MM-dd hh:mm:dd。Freezes will use the file lastupdte date if this meta not found.
layout Specified the layout page file name without extension that in templates/layouts
tags Tags. If you need to specified multiple tags you should use comma separated.
id The position index.

Recommendations

  • You should add tags,summary to your documents becasuse they will rendered in <meta> of the <output page that can be enhanced the SEO。
  • In order to sort the articles by datetime you should add date meta.
  • tags should be accurate and do not use over 10 tags.

The Freezes makes freezes command tools to run or build your website.

Run Freezes

Freezes will run a self-host server and open to the default browser after freezes executed.

Freezes loads a livereload server that can be auto reload the browser page when contents change.

(venv) my-site$ freezes

Transform to static website

In order to get high performances when you writing posts/pages , design web styles or customize layouts Freezes disable the auto static page generation feature.

So when you need to build your static website manually :

(venv) my-site$ freezes build

After the command run. The website is build to builds folder under current working folder.

Force assets rebuild

(venv) my-site$ freezes assets --build

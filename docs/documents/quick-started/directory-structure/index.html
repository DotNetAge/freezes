Freezes is a document-driven website and a text transformation engine.

A basic Freezes site usually looks something like this:

.
├── _config.yml
├── _assets.yml
├── pages
|   ├── posts
|   |     └── 2014-01-01
|   |           ├─ why-programmer-should-write-books.md
|   |           └─ why-programmer-should-write-books
|   ├── documents     
|   |   
|   └── index.md
├── builds
├── static
|   ├── images
|   ├── scripts
|   └── styles
├── templates
|   ├── includes
|   ├── layouts
|   └── 404.html
└── __init__.py

An overview of what each of these does:

FILE / DIRECTORY DESCRIPTION REMARKS
pages Document folder Save all markdown files
pages/posts Posts folder
pages/index.md Home page
static Resources All static resource files
static/images Images
static/scripts Scripts
static/styles Styles
templates Templates
templates/includes Includes
templates/layouts Layouts
builds Build Build result files
_config.yml Site configuration
_assets.yml Asset configuration

Quick install

Freezes offers the simple way to setup the runtime environment by setup script.

Download one of setup script below

  • Freezes setup script for Linux (linux-install.sh)
  • Freezes setup script for OSX (mac-linstall.sh)

and runt the command below in your terminal:

Ubuntu 

$ . linux-install.sh

OSX 

$ . mac-install.sh

that's all!

Installation Manual

You can also install Freeze by manual.

Run the commands below in your terminal:

$ virtualenv venv
$ source venv/bin/activate
(venv)$ pip install freeze
(venv)$ mkdir my-website && cd $_
(venv)$ freezes new
(venv)$ freezes

That’s nothing, though. The real magic happens when you start creating blog posts, using the front matter to control templates and layouts, and taking advantage of all the awesome configuration options Frezzes makes available.

If you’re running into problems, ensure you have all the requirements installed.

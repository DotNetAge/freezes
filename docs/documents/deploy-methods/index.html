Sites built using Freezes can be deployed in a large number of ways due to the static nature of the generated output. A few of the most common deployment techniques are described below.

Web hosting providers (FTP)

Just about any traditional web hosting provider will let you upload files to their servers over FTP. To upload a Freezes site to a web host using FTP, simply run the freezes build command and copy the generated builds folder to the root folder of your hosting account.

SSH

$ tar -czvf builds.tgz build
$ ssh server@example.com
$ scp builds.tgz server@example.com:/var/www/my-site
$ mkdir /var/www/my-site
$ tar -xzvf builds.tgz

Github

$ git add . 
$ git commit -m "Initial commit"
$ git remote add origin https://github.com/username/my-freezes-site.git
$ git push origin gh-pages

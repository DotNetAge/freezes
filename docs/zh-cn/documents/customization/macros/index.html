Freezes 提供了少量很实用的页面宏，辅助您快速输出某些特定功能到页面中，在将来版本还会继续增加更多的预定宏，如果您有什么更好的可包装为宏的功能请在github上给我留言

comment

comment宏包装了 Disqus 的页面讨论功能。

用法

{% import '_macros/disgus.html' as disqus %}

{{ disqus.comment('你在Disqus注册的用户名')}}

list

显示 页面/文章 对象列表

输入参数说明

参数 类型 说明
post_items page对象列表 显示的页面/文章列表
excludes 字符串列表 (可选)不显示于列表中的页面的 path 属性值

用法

{% import '_macros/posts.html' as posts %}

{{ posts.list(site.query(page.path))}}

gallery

将页面/文章以图示列表方式输出显示。

输入参数说明

参数 类型 说明
post_items page对象列表 显示的页面/文章列表
excludes 字符串列表 (可选)不显示于列表中的页面的 path 属性值

用法

{% import '_macros/posts.html' as posts %}

{{ posts.gallery(site.query(page.path))}}

tags

输入站点内所有的主题标签列表

用法

{% import '_macros/posts.html' as posts %}

{{ posts.tags()}}

relatives

输出与指定文章相关的其它的文章的链接列表

输入参数说明

参数 类型 说明
post page 参考页面对象

用法

{% import '_macros/posts.html' as posts %}

{{ posts.relatives(page)}}

crumb

输入参数说明

参数 类型 说明
site site 输入当前页面上下文中的site对象
_path 字符串 用于查询的页面路径 page对象的path属性

用法

{% import '_macros/posts.html' as posts %}

{{ posts.crumb(site,page.path)}}

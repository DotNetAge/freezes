当你在定制布局页时必不可少地需要从Freezes中获取一些数据显示在页面中，本文将详细地列出在布局页内所使用使用的所有的变量对象以及成员的使用说明。

站点 - Site

特性说明

名称 类型 说明
pages Page 列表 返回当前站点内所有页面
posts Page 列表 返回当前站点内所有的博客文章
data Object 返回数据区加载的对象引用
tags 字符串 字典 返回当前站点内所有的主题标签
archives Archive 列表 返回当前站点内所有的归档对象

方法参考

menus(search_path='') 方法

查询指定参考路径下用于显示于菜单的所有页面的对象集合。

参数

  • search_path - 字符串 指定参考路径

返回值

返回当前参考路径下的所有的子页面对象集合

get_pages_path(context_path) 方法

根据指定上下文路径返回该路径下的页面列表

参数

  • context_path - 字符串 用于查询的上下文路径

返回值

页对象列表

locale_posts(locale='') 方法

获取指定语言的所有博客文章

参数

  • locale - 字符串 指定语言

返回值

页对象列表

query(search_path,all=False,sort='id') 方法

查的指定路径下所有的页面

参数

  • search_path - 字符串 指定查询路径
  • all - 布尔值 当为真是反回路径下所有的子代页
  • sort - 字符串 指定排序用的元数据

返回值

页对象列表

tagged(tag,locale='') 方法

返回指定 主题词 的所有页面列表

参数

  • tag - 字符串 指定参考路径
  • locale - 字符串 指定语言

返回值
页面列表对象

页 - Post

特性说明

名称 类型 说明
title 字符串 页标题文字
summary 字符串 摘要
meta 字典 元数据字典
excerpt 字符串 概述
relatives Page 列表 相关文章列表
next Page 下一页对象
prev Page 上一页对象
url 字符串 URL
name 字符串 页文件名（不包含扩展名）
wordcount 整数 字数
created 日期 创建日期
last_updated 日期 最后修改日期
published 日期 发表日期

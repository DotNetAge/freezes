Freezes 默认安装后会提供一个简约的网页风格。每个文稿可以通过元数据 layout 指定当前文稿用什么页面布局显示。默认情况下当文稿内没有采用 layout元数据指定布局时， Freezes 将为你的文稿自动选择一种最合适的布局方式。

Freezes 提供了以下的3种布局方式:

布局 说明
recent 上方显示正文，下方显示最近站点在 pages/posts 目录中最新的文章
page 上方显示正文，下方显示子目录下的文章
post 上方显示下方，底部带有讨论功能

具体指定的方法可以在文稿内增加layout，如下所示：

layout: recent
title: "关于"

此处是正文

如果你是一位网页设计师，又或者你是一位前端开发者，而且当你需要定义自己的布局方式，那么你可以阅读以下的文章：

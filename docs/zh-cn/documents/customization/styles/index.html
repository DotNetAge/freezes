自定义样式

Freezes 默认搭载 bootstrap 和 fontawesome , 默认支持 Less 作为样式语言 static/styles 下的样式文件将会被自动编译并合成、压缩为一个 css 自动引入到页面内。

你可以修改 static/styles/site.less 内的样式效果，或者选择直接使用 bootstrap 提供标准样式类。对此 Freezes 并没有作出任何的限制，只要你喜欢你甚至可以将所的默认布局方式改掉换成你所需要的样子。

在 static/styles/ 目录下的所有*.less文件都会进行合并编译，如果要增加新的样式引用，可以在该目录下增加你的子目录，然后打开 _assets.yml 配置文件增加新的编译配置。

如果你认为这样做很复杂，那么你可以直接在 static 目录内直接放置 css 文件（当然我们并不建议这样做），然后在页面布局中用 <link> 标引入该文件：

范例：假如在 static 目录中增加了一个 my_style.css 的文件，那么就可以在需要引用该样式的布局内加入以下代码：

范例布局 templates/layouts/my_page_layout.html

{% extends "_layouts/_page.html" %}
<!--此处是增加对my_style.css文件的引用-->
{% block styles %}
    {{ super() }}
    <link ref="stylesheet" 
          type="text/css" 
          href="{{ url_for('static',filename="my_style.css") }}" />
{% endblock%}
<!---->

本例中的 {{ super() }} 是指使用Freezes母板内的原有样式，如果不加上此标记所有的样式将不会被添加到当前布局页。

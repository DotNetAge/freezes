Freezes 的配置项非常简单，只是用于收集一些站点相关的文字信息和你的一些个人联系信息，在站点目录下的 _config.yml 为站点配置文件，用文本编辑工具打开，内容如下:

title: "网站显示的标题文字" 
email: your_mail@email.com
description: "网站的简单描述"
url: "http://www.domain.com"
twitter: your-twitter-username
github:  your-github-username
disqus_shortname: your-disqus-website-shortname
ga_account: your-ga-accocunt
lang: en
配置项 说明
title * 设置网站所显示的标题
email 设置您的联系邮件,如果设置了则显示于站点的联系信息栏
description 网站的基本描述 (可选)
url * 静态化后站点部署时使用的url 例：http://freezes.dotnetage.com 。此选项必须正确设置否则站点被静态化后可能会出现局布的页面链接错误。
twitter 设置 twitter 上的联系用户名 (可选)
github 设置 github 上的用户名 (可选)
disqus_shortname 设置 disqus 的注册站点名称，只有设置后 文章评论才可用
ga_account 设置google分析的帐号名，设置后将自动在每页底部加上 google 分析的代码
lang 默认语言, 如果不设定则为 'en'

当配置项内容被更改后必须重新启动 Freezes 更改才能被应用，中文请使用 ""。

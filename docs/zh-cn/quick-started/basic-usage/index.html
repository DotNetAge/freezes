Freezes 是一个基于 python 编写的命令行工引导工具。如果您正在使用Linux或者OSX，那么恭喜你！你的平台中已经预装有 python 的运行环境，可以直接 freezes 的指令。如果您是一位Windows的用户，那么还请先点击下载 python 2.7

创建站点

创建一个目录并使用 new 参数运行 freezes，Freezes 将自动在当前中为你建立站点所需要目录和文件。

(venv) my-site$ freezes new

运行站点

直接执行 freezes 指令将会运行一个自带的Web服务，并会在您的默认浏览器内自动访问该本地网站地址(默认为:http://localhost:5500)。

(venv) my-site$ freezes

Freezes 带有自动加载更新的功能，也就是说如果当前目录内的任何文件发生了变化，被Freezes打开的浏览器页面都会执行自动刷新。如此一来，无论你是在写作还是在做设计都可以即时看到最终的输出效果而无需要进行人工刷新。

生成静态化站点

当完成了所有的写作或者设计工作准备将站点部署到外部服务器，那么请在终端内运行以下指令：

(venv) my-site$ freezes build

指令执行完成后，当前站点的所有的输出页面都被静态化为html并放置于 builds 目录下，将此目录下的全部文件直接拷贝至您的Web服务器就可完全所有的部署工作了。

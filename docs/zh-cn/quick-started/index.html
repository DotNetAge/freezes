快捷安装

如果你的机器完全没有配置 python 环境和相关的应用包，那么可以使用我们提供的快速安装脚本，安装脚本将为你将支持环境和Freezes一并安装。

安装脚本：

  • Freezes for Linux 安装脚本 (linux-install.sh)
  • Freezes for OSX 安装脚本 (mac-linstall.sh)

下载后在终端直接运行:

Ubuntu 

$ . linux-install.sh

OSX 

$ . mac-install.sh

Freezes 的使用就这么简单！当你开始写作或者进行设计时你将会更加深层地体验到她为您带来的方便。

手工安装

如果你是个指令控，那么也可以在您的电脑 终端程序 键入以下的命令：

$ virtualenv venv
$ source venv/bin/activate
(venv)$ pip install freeze
(venv)$ mkdir my-website && cd $_
(venv)$ freezes new
(venv)$ freezes

但需要先安装 Freezes 的运行依赖环境

如果在安装过程中出现了问题，请参考安装要求。
